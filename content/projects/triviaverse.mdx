## Key decisions

The interface establishes a common rhythm across different play modes before realtime complexity is introduced. That makes the product language testable without presenting backend contracts as working multiplayer.

## Architecture

Expo and React Native provide the mobile foundation. Supabase schemas and contracts describe future room state, shared rounds, profiles, progression and leaderboards while implementation continues.
