## Key decisions

Wholesale language is embedded in the interaction model: users can compare tax treatments, respect order increments and jump to pallet quantities without translating from a consumer storefront.

The catalogue admin validates imports before committing browser-side data and can export it again, making the prototype useful for shaping a real back-office workflow without pretending localStorage is a production database.

## Architecture

The product is intentionally a static React SPA. Account terms, checkout and persistence are mock or browser-local flows that make the buying model testable before a secure commerce backend is introduced.
