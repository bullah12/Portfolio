## Key decisions

Workspace isolation is enforced as a cross-cutting rule and covered in tests, rather than applied inconsistently at the page layer. Ownership is represented as an effective-dated event ledger, so corrections reverse or supersede history instead of erasing it.

Private ownership payments and property-fund capital flows use separate accounting treatments. That distinction keeps investment performance legible as the domain grows.

## Architecture

Next.js and Prisma sit over PostgreSQL and Supabase. TanStack tools support dense, interactive operations, while focused tests cover isolation, ownership, leases, income and portfolio performance.
