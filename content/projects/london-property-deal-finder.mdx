## Key decisions

Every ingestion adapter preserves source identity and freshness timestamps. Upserts use listing identity so repeat runs update a property rather than creating an increasingly noisy duplicate trail.

Browser rendering is a fallback, not the default: direct requests remain the faster path wherever a source permits them. Same-day skip and force controls make scheduled automation safer to operate and easier to investigate.

## Architecture

Python adapters normalise incoming listings before Supabase persistence. Streamlit provides focused review surfaces for recent and complete inventories, protected by a password while the product remains private.
