## Fuel optimisation

- Built an explainable arrival-delay fuel-optimisation solution end to end.
- Implemented silver and gold medallion layers in an AWS Databricks sandbox.
- Trained and registered a hurdle model in MLflow and created a serving endpoint for delay inference.
- Built a Power BI drill-through experience for planner decisions.
- Engineered resilient Meteostat and AviationWeather ingestion with schema-variability handling.

## AMIE — Aircraft Maintenance Insights Engine

- Served as data lead for an MVP that grouped semantically similar aircraft defects and surfaced recurring issues.
- Built medallion pipelines with multi-source ingestion, standardisation, de-duplication, cleaning and PII masking.
- Applied embeddings and K-Means clustering with simple labels and trend summaries.
- Delivered a Streamlit exploration dashboard with a feedback loop for improving clusters and labels.
- Operationalised parameterised jobs, refresh schedules, Azure Repos CI/CD, quality checks and governance documentation.

## Enterprise Data Quality / DQX

- Designed a reusable Silver Layer standard.
- Built and optimised Azure Data Factory validation workflows using Azure Repos and notebooks for CI/CD and collaboration.
- Raised issues with the open-source project and promoted automated quality and governance practices.
